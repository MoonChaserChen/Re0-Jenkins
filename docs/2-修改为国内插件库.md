# 修改为国内插件库
## 使用国内插件库：
	版本：jenkins-windows-2.204.5
	
0. 不使用国内插件库安装插件会很慢，甚至失败
1. 安装目录下修改配置文件: updates\default.json
    替换 updates.jenkins-ci.org/download 为 mirrors.tuna.tsinghua.edu.cn/jenkins
    将connectionCheckUrl替换为 http://www.baidu.com/
2. 重启
